## Tips

- **Format:** JPEG for photos, PNG for logos/graphics with transparency, WebP for best compression.
- **Size:** Keep hero images under 400 KB, gallery/thumbnails under 150 KB. Use [Squoosh](https://squoosh.app) to compress.
- **After uploading:** The HTML files already reference these filenames — images will appear automatically once placed here with the correct names.
